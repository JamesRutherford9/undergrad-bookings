Main Page

This is the main page for the booking application.

  • The view bookings tab allows you to see all of you personal bookings.
  • The create booking tab brings up the grid for creating a booking. Click an empty cell to start the booking process.
  • The chatbot tab is a page allowing interaction with an Azure chatbot, which should answer common questions about the bookings.
  • Finally, the today's bookings tab shows a list of bookings taking place today. This gives access to the ongoing booking interaction pages.
